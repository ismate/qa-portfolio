# SQL_Practice

Репозиторий для практики SQL-запросов.  
Включает задания на SELECT, JOIN, агрегатные функции, GROUP BY, HAVING, ORDER BY, а также изменение данных (INSERT, UPDATE, DELETE).

---

## 📚 Структура репозитория

- `01_select_where.sql` — базовые запросы SELECT с условиями WHERE  
- `02_join_count.sql` — JOIN таблиц, агрегатные функции COUNT, SUM, AVG  
- `03_order_by_having_group_by.sql` — группировка, фильтрация агрегатов и сортировка  
- `04_insert_update_delete.sql` — добавление, изменение и удаление данных (INSERT / UPDATE / DELETE)  
- `05_combined_practice.sql` — комбинированная практика всех изученных тем  

---

## 📝 Задачи

1. **SELECT + WHERE**  
   - Практика фильтрации данных по условиям  

2. **JOIN + COUNT/SUM/AVG**  
   - Соединение таблиц  
   - Подсчет количества, суммы и среднего значений  

3. **GROUP BY + HAVING + ORDER BY**  
   - Группировка данных по колонкам  
   - Фильтрация агрегатных данных  
   - Сортировка результатов  

4. **INSERT / UPDATE / DELETE**  
   - Добавление новых записей  
   - Изменение существующих данных  
   - Удаление данных  

5. **Комбинированная практика**  
   - Объединение всех изученных тем для закрепления навыков
