## TC-003-add-to-befree-cart
**Название: Проверка добавления товара в корзину неавторизованным пользователем**

**Приоритет: High** 
**Тип теста: Functional**  

**Предусловия:**
- Открыта страница каталога: https://befree.ru/muzhskaya
- Пользователь не авторизован

**Тестовые данные:**
- Категория: Мужская одежда
- Размер: XS

**Шаги:**
1. Открыть страницу каталога
2. Открыть навигационное меню (бургер)
3. Перейти в категорию "Одежда"
4. Выбрать любой товар из списка
5. Выбрать доступный размер
6. Нажать кнопку "Добавить в корзину"

**Ожидаемый результат:**
- Товар добавлен в корзину
- В корзине отображается выбранный товар
- Отображается корректная цена товара
- Отображается выбранный размер
- Количество товара = 1

**Фактический результат:**
- Товар успешно добавлен в корзину
- Цена, размер и количество соответствуют выбранным значениям

**Статус: Passed**

**Окружение:**
- Chrome 146 / Windows 11 / PC
