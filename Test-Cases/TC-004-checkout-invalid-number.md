## TC-004-checkout-invalid-number
**Название: Проверка валидации номера телефона при оформлении заказа**

**Приоритет: High** 
**Тип теста: Negative**  

**Предусловия:**
- Открыта страница каталога: https://www.lamoda.ru/women-home/
- Пользователь не авторизован
- Товар доступен для заказа

**Тестовые данные:**
- Имя: Test
- Фамилия: Test
- Номер телефона: +7 (123) 456-78-90 (несуществующий номер)

**Шаги:**
1. Открыть страницу каталога
2. В навигационном меню перейти в раздел "Одежда"
3. Нажать на любой товар в предложенных вариантах
4. Выбрать размер
5. Нажать кнопку "Добавить в корзину"
6. Перейти в корзину
7. Нажать кнопку "К оформлению"
8. Выбрать любой Пункт выдачи
9. Заполнить данные для оформления заказа
10. Нажать кнопку "Оформить заказ"

**Ожидаемый результат:**
- Указанный номер телефона не проходит валидацию
- Клиент получает сообщение об ошибке : "Укажите валидный номер телефона"
- Оформление заказа не выполняется

**Фактический результат:**
- Поле принимает невалидный номер телефона
- Ошибка валидации не отображается
- Система отправляет код подтверждения
- Код не доставляется, оформление невозможно завершить

**Статус: Failed**

**Окружение:**
- Chrome 146 / Windows 11 / PC
