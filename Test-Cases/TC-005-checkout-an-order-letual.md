## ID: TC-005-checkout-an-order-letual
**Название: Проверка оформления заказа с авторизацией пользователя**

**Приоритет: High** 
**Тип теста: Functional**  

**Предусловия:**
- Открыта главная страница сайта : https://www.letu.ru/
- Пользователь не авторизован

**Тестовые данные:**
- Номер телефона: +79999999999(тестовый номер)
- Способ оплаты: любой доступный
- Пункт выдачи: любой доступный

**Шаги:**
1. Нажать кнопку "Каталог"
2. Перейти в раздел "Парфюмерия"
3. Выбрать и нажать на любую карточку товара
4. Добавить товар в корзину
5. Перейти в корзину
6. Нажать кнопку продолжить
7. Пройти авторизацию
8. Выбрать магазин для получения
9. Выбрать способ оплаты
10. Нажать кнопку "Оплатить"
11. В открывшейся форме заполнить данные (номер телефона,имя,email)
12. Нажать кнопку "Оплатить"


**Ожидаемый результат:**
- Пользователь успешно авторизуется
- Товар отображается в заказе
- Цена и количество соответствуют выбранным значениям
- Данные пользователя сохраняются
- Заказ успешно оформляется  

**Фактический результат:**
- Пользователь успешно авторизуется
- Заказ успешно оформляется
- Цена и количество товара соответствуют выбранным значениям  

**Статус: Passed**

**Окружение:**
- Chrome 146 / Windows 11 / PC
