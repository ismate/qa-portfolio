# BUG-018: Смещение кнопки Checkout при оформлении заказа у пользователя visual_user

Окружение:
- Браузер: Chrome 145.0.7632.109
- ОС: Windows 11
- Устройство: PC
- Сайт: https://www.saucedemo.com/
- Пользователь: visual_user

Предусловия:
- Пользователь не авторизован

Шаги воспроизведения:
1. Открыть сайт https://www.saucedemo.com/
2. Ввести логин: visual_user
3. Ввести пароль: secret_sauce
4. Нажать кнопку "Login"
5. Нажать на иконку корзины в правом верхнем углу

Фактический результат:
- На странице "Cart" кнопка "Checkout" отображается некорректно
- Кнопка расположена вплотную к правому верхнему углу экрана

Ожидаемый результат:
- Кнопка Checkout должна находиться в основном блоке оформления заказа
- Кнопка должна быть хорошо видна и корректно выровнена по UI

Серьезность: Значительная (UI)  
Приоритет: Средний  
Воспроизводимость: 5/5
