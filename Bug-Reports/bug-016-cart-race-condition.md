# BUG-016: Непредсказуемое поведение корзины при добавлении товара и обновлении страницы

Окружение:
- Браузер: Chrome 145.0.7632.109
- ОС: Windows 11
- Устройство: PC
- Сайт: https://www.detmir.ru/

Предусловия:
- Пользователь на сайте
- Открыта корзина с блоком рекомендуемых товаров

Шаги воспроизведения:
1. Перейти на сайт https://www.detmir.ru/
2. В правом верхнем углу открыть корзину
3. В блоке рекомендуемых товаров добавить 3 шт одного товара
4. Быстро удалять и/или добавлять один и тот же товар несколько раз
5. Немедленно обновить страницу

Фактический результат:
- После обновления страницы количество товаров отображается некорректно:
  - Удалённые товары возвращаются
  - Добавленные товары не учитываются

Ожидаемый результат:
- Количество товаров в корзине должно корректно сохраняться и обновляться независимо от скорости действий пользователя

Серьезность: Незначительная (UX)  
Приоритет: Низкий  
Воспроизводимость: Intermittent (зависит от скорости действий)
