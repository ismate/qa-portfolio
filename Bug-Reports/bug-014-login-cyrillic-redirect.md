# BUG-014: Отсутствие ошибки и редирект на другую страницу при вводе кириллицы в логин/пароль

Окружение:
- Браузер: Chrome 145.0.7632.109
- ОС: Windows 11
- Устройство: PC
- Сайт: https://bolshoi.ru/

Предусловия:
- Пользователь на сайте
- Форма входа доступна

Шаги воспроизведения:
1. Открыть сайт https://bolshoi.ru/
2. Открыть меню
3. Нажать на личный кабинет
4. Нажать кнопку “Войти”
5. Ввести кириллицу в поля логина и пароля (например: тест)
6. Нажать кнопку “Войти”

Фактический результат:
- Страница обновляется, происходит переход в раздел с билетами
- Введённые данные очищаются
- Сообщение об ошибке не отображается

Ожидаемый результат:
- Должно отображаться сообщение об ошибке (например: «Неверный логин или пароль» или «Допустимы только латинские символы»)
- Пользователь должен оставаться на форме входа без редиректа и перезагрузки страницы

Серьезность: Значительная  
Приоритет: Средний  
Воспроизводимость: 5/5
