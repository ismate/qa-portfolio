# BUG-005: Возможность оформления заказа с пустой корзиной у пользователя standard_user

Окружение:
- Браузер: Chrome 145.0.7632.109
- ОС: Windows 11
- Устройство: PC
- Сайт: https://www.saucedemo.com/inventory.html
- Пользователь: standard_user

Предусловия:
- Пользователь зарегистрирован на сайте
- Пользователь авторизован под standard_user

Шаги воспроизведения:
1. Открыть сайт https://www.saucedemo.com/inventory.html
2. Авторизоваться под standard_user / secret_sauce
3. Перейти в корзину
4. Не добавляя товары, нажать кнопку "checkout"
5. Заполнить данные адреса
6. Нажать кнопку "continue"
7. Нажать кнопку "finish"

Фактический результат:
- Заказ создается без товаров в корзине

Ожидаемый результат:
- При попытке оформить заказ с пустой корзиной должно появляться сообщение об ошибке
- Заказ не должен создаваться

Серьезность: Незначительная  
Приоритет: Низкий  
Воспроизводимость: Всегда
