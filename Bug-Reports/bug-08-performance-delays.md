# BUG-008: Задержки при совершении действий на сайте

Окружение:
- Браузер: Chrome 145.0.7632.109
- ОС: Windows 11
- Устройство: PC
- Сайт: https://www.saucedemo.com/inventory.html
- Пользователь: performance_glitch_user

Предусловия:
- Пользователь зарегистрирован на сайте
- Пользователь авторизован под performance_glitch_user

Шаги воспроизведения:
1. Открыть сайт https://www.saucedemo.com/inventory.html
2. Авторизоваться под performance_glitch_user / secret_sauce
3. Обновить страницу, зайти в корзину, поменять фильтры
4. Нажать на значок панели в левом верхнем углу
5. В панели перейти по вкладке all items

Фактический результат:
- При авторизации, обновлении страницы, переходе в корзину, смене фильтров, нажатии на вкладку all items происходит задержка около 6 секунд

Ожидаемый результат:
- При авторизации, обновлении страницы, переходе в корзину, смене фильтров, нажатии на вкладку all items задержки не должно быть

Серьезность: Критическая  
Приоритет: Средний  
Воспроизводимость: Всегда
