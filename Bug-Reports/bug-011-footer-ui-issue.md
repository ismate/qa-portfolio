# BUG-011: UI ошибка с футером

Окружение:
- Браузер: Chrome 145.0.7632.109
- ОС: Windows 11
- Устройство: PC
- Сайт: https://www.demoblaze.com/

Предусловия:
- Пользователь на сайте

Шаги воспроизведения:
1. Открыть сайт https://www.demoblaze.com/
2. Зайти в корзину
3. Убедиться, что на странице мало контента
4. Обратить внимание на футер

Фактический результат:
- На страницах с маленьким количеством контента футер плавает выше низа, под ним остаётся белое пространство

Ожидаемый результат:
- Футер должен быть прижат к низу экрана

Серьезность: Тривиальная  
Приоритет: Низкий  
Воспроизводимость: Всегда
