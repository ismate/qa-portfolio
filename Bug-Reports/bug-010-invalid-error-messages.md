# BUG-010: Некорректный вывод ошибок при заполнении полей для ввода

Окружение:
- Браузер: Chrome 145.0.7632.109
- ОС: Windows 11
- Устройство: PC
- Сайт: https://www.saucedemo.com/inventory.html
- Пользователь: standard_user

Предусловия:
- Пользователь зарегистрирован на сайте
- Пользователь авторизован под standard_user

Шаги воспроизведения:
1. Открыть сайт https://www.saucedemo.com/inventory.html
2. Авторизоваться под standard_user / secret_sauce
3. Зайти в корзину
4. Нажать checkout
5. Оставить все поля пустыми и нажать Continue

Фактический результат:
- Все три поля подсвечиваются красным, но отображается только одно сообщение об ошибке:
  - Сначала: "Error: First Name is required"
  - После заполнения First Name: "Error: Last Name is required"
  - После заполнения Last Name: "Error: Postal Code is required"

Ожидаемый результат:
- При всех пустых полях должны отображаться три сообщения об ошибке одновременно, чтобы пользователь видел все обязательные поля

Серьезность: Незначительная  
Приоритет: Низкий  
Воспроизводимость: Всегда
