# 🐞 Ошибка "Что-то пошло не так" при оформлении заказа без интернета

**Окружение:**
- Браузер: Chrome 145.0.7632.109
- ОС: Windows 11
- Устройство: PC
- URL: https://www.detmir.ru/

**Предусловия:**
- Пользователь авторизован
- Корзина заполнена
- Обязательные поля оформления заказа заполнены

**Шаги воспроизведения:**
1. Перейти на сайт https://www.detmir.ru/
2. Перейти в корзину
3. Нажать кнопку "оформить заказ"
4. Нажать F12 → открыть вкладку Network
5. Включить offline (симуляция отсутствия интернета)
6. Выбрать пункт самовывоза

**Фактический результат:**
- Появляется сообщение "Что-то пошло не так"
- Сайт не перезагружается
- Пользователь не понимает, что делать дальше

**Ожидаемый результат:**
- Появляется понятная ошибка: "Проверьте соединение с интернетом"
- Появляется кнопка "Обновить" или "Попробовать снова"
- Пользователь может повторить попытку без дублирования действий

**Серьезность:** Незначительная  
**Приоритет:** Средняя  
**Воспроизводимость:** Всегда (при отключенном интернете)
