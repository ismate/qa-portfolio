# Тестирование совместимости сайта SauceDemo на разных версиях Android
# 🐞 Белый экран при переходе в корзину (Android 16)

## 📋 Заголовок
При переходе в корзину на сайте SauceDemo в эмуляторе с Android 16 открывается пустой белый экран. Контент не загружается, дальнейшая навигация невозможна.

---

## 🔍 Окружение

| Параметр | Значение |
|----------|----------|
| **Устройство** | Эмулятор Pixel 9 (Android Studio AVD) |
| **ОС** | Android 16 (API 36) |
| **Браузер** | Google Chrome 146 |
| **Разрешение экрана** | 1080 × 2400 пикселей |
| **Сайт** | https://www.saucedemo.com |
| **Пользователь** | `problem_user` / `secret_sauce` |
| **Дата обнаружения** | март 2026 |

---

## 🔄 Шаги воспроизведения

1. Запустить эмулятор с Android 16
2. Открыть браузер Chrome
3. Перейти на сайт https://www.saucedemo.com
4. Ввести логин: `problem_user`
5. Ввести пароль: `secret_sauce`
6. Нажать кнопку **Login**
7. Нажать на иконку корзины в правом верхнем углу

---

## 🚫 Фактический результат

Вместо страницы корзины открывается **пустой белый экран**. Сайт перестаёт реагировать на любые действия (клики, скролл). Перезагрузка страницы не помогает.

---

## ✅ Ожидаемый результат

Должна открыться страница корзины с корректным отображением:
- Сообщение **"Your cart is empty"**, если товары не добавлялись
- Список добавленных товаров с названиями, ценами и количеством
- Кнопка **"Checkout"** для оформления заказа
- Возможность вернуться к покупкам

## 🧪 Дополнительное тестирование

| Версия Android | Результат |
|----------------|-----------|
| Android 11 | ✅ Работает |
| Android 13 | ✅ Работает |
| Android 14 | ✅ Работает |
| **Android 16** | ❌ Белый экран |

**Вывод:** Баг специфичен для Android 16 и не воспроизводится на более ранних версиях.

---

## 💡 Возможная причина

Вероятно, сайт SauceDemo разрабатывался в период Android 11–14 и не был адаптирован для новых версий ОС. На Android 16 происходит сбой при обработке перехода на страницу `/cart.html`.

---

- Баг воспроизводится стабильно на разных эмуляторах с Android 16
- Требуется проверка на реальном устройстве с Android 16 (при наличии)
