# BUG-017: Смещение иконки корзины на странице товаров при авторизации под visual_user

Окружение:
- Браузер: Chrome 145.0.7632.109
- ОС: Windows 11
- Устройство: PC
- Сайт: https://www.saucedemo.com/
- Пользователь: visual_user

Предусловия:
- Пользователь не авторизован

Шаги воспроизведения:
1. Открыть сайт https://www.saucedemo.com/
2. Ввести логин: visual_user
3. Ввести пароль: secret_sauce
4. Нажать кнопку "Login"

Фактический результат:
- На странице Inventory, иконка корзины отображается со смещением относительно хедера
- Иконка выходит за пределы своей позиции в layout

Ожидаемый результат:
- Иконка корзины должна корректно отображаться в правом верхнем углу хедера и быть выровненной по UI-дизайну

Серьезность: Незначительная (UI)  
Приоритет: Низкий  
Воспроизводимость: 5/5
