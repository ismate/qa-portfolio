# 🐞 При оформлении заказа во второй вкладке используется устаревшее состояние корзины

**Окружение:**
- Браузер: Chrome 145.0.7632.109
- ОС: Windows 11
- Устройство: PC
- URL: https://euroauto.ru

**Предусловия:**
- Пользователь авторизован на сайте

**Шаги воспроизведения:**
1. Перейти на сайт https://euroauto.ru
2. В навигации нажать вкладку "Шины"
3. Выбрать 2 разные позиции
4. В карточке товара нажать кнопку "Купить"
5. Перейти в корзину
6. В новой вкладке открыть корзину еще раз
7. На 1-й вкладке добавить по 1-2 единицы товара
8. На 2-й вкладке заполнить информацию о контактах получателя
9. Указать пункт выдачи заказов на карте
10. Нажать кнопку "Оформить заказ"

**Фактический результат:**
- Заказ оформляется с количеством товара, актуальным для второй вкладки (устаревшее состояние)
- Изменения, внесённые в первой вкладке, в заказ не попадают

**Ожидаемый результат:**
- При оформлении заказа должна использоваться актуальная версия корзины, сохранённая на сервере, независимо от вкладки

**Серьезность:** Критический (потеря данных заказа)  
**Приоритет:** Высокий  
**Воспроизводимость:** Всегда
