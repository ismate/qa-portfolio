# BUG-013: Возможность регистрации с некорректным количеством символов в поле Username

Окружение:
- Браузер: Chrome 145.0.7632.109
- ОС: Windows 11
- Устройство: PC
- Сайт: https://www.demoblaze.com/

Предусловия:
- Пользователь на сайте
- Форма регистрации доступна

Шаги воспроизведения:
1. Открыть сайт https://www.demoblaze.com/
2. В форме регистрации в поле Username ввести более 15 символов
3. Ввести пароль
4. Нажать кнопку регистрации (Sign up)

Фактический результат:
- Ошибка не возникает при вводе более 15 символов
- При успешной регистрации с длинным именем пользователя съезжает верстка

Ожидаемый результат:
- При вводе более 15 символов должна выводиться ошибка
- Регистрация с таким именем пользователя должна быть ограничена

Серьезность: Тривиальная  
Приоритет: Низкий  
Воспроизводимость: Всегда
