# BUG-002: Кнопка Add to cart не работает для некоторых товаров для пользователя problem_user

**Окружение:**
- Браузер: Google Chrome 145.0.7632.109
- ОС: Windows 11
- Устройство: PC
- Сайт: https://www.saucedemo.com/inventory.html
- Пользователь: problem_user

**Предусловия:**
- Пользователь зарегистрирован на сайте
- Пользователь авторизован под problem_user

**Шаги воспроизведения:**
1. Перейти на страницу товаров: https://www.saucedemo.com/inventory.html
2. Авторизоваться под problem_user / secret_sauce
3. На странице с товарами попытаться добавить следующие товары в корзину:
   - Sauce Labs Bolt T-Shirt ($15.99)
   - Sauce Labs Fleece Jacket ($49.99)
   - Test.allTheThings() T-Shirt (Red) ($15.99)

**Фактический результат:**
- Нажатие на кнопку **Add to cart** не добавляет товары в корзину
- Количество товаров над значком корзины не обновляется
- Кнопка **Add to cart** не меняет состояние на **Remove** для всех трёх товаров

**Ожидаемый результат:**
- При нажатии на кнопку **Add to cart** товары добавляются в корзину
- Цифра над значком корзины увеличивается на 1
- Кнопка меняется на **Remove** для выбранного товара

**Серьезность:** Средняя  
**Приоритет:** Средний  
**Воспроизводимость:** Всегда
