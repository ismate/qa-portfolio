# BUG-015: Кнопка смены языка не работает при быстром многократном нажатии на разных страницах сайта

Окружение:
- Браузер: Chrome 145.0.7632.109
- ОС: Windows 11
- Устройство: PC
- Сайт: https://bolshoi.ru

Предусловия:
- Пользователь на сайте

Шаги воспроизведения:
1. Перейти на сайт https://bolshoi.ru
2. На главной странице быстро нажать на кнопку смены языка несколько раз
3. Перейти на вкладку “Сезон 2025-2026”
4. Быстро нажать на кнопку смены языка несколько раз
5. Повторить действия на всех вкладках навигации

Фактический результат:
- При быстром многократном нажатии стили при наведении остаются активными, но текст интерфейса не меняется
- Язык можно переключить только после перезагрузки страницы

Ожидаемый результат:
- Язык должен переключаться вне зависимости от частоты и скорости нажатия на кнопку, на всех страницах сайта

Серьезность: Незначительная  
Воспроизводимость: Intermittent (зависит от скорости нажатий и страницы)
